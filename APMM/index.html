APMM

Course Description

经济学作为国际留学非常热门的选择，一直是大众之选。其他的无需多说，但是其课程量和作业量相对较大。

授课老师有两位。
一位是AP两大元老之一，在十一教了12年的的Mr. Robert Rush，上课很有意思，讲的内容更intuitive一点，更丰富一点。不像是那种一板一眼按照课本教的。但是Rush自己写了一本双经的课本，所以也可以说他是按自己写的课本一板一眼教的LOL。但是对于这位老师的评价存在争议，有的同学认为他风趣幽默有的同学认为他不讲AP考试知识而且容易情绪化发怒。
另外一位是今年新来的ED Eduardo，巴西阳光男孩（不是），人很好，而且人非常认真，但是因为是在巴西远程线上上课，所以效果不是很好。

GPA的话相对来说并不是很好拿，个人仅了解Rush班，高二上是一共4个A+，剩下的1/2 A，1/4B和1/4C。

Comments

如果你在高二一开学被分配到了Rush教的双经班，赶紧调班吧。这个老师虽然很有意思，但是缺点（情绪问题，GPA制度等等）真的不能忍受。

  1. 情绪极度不正常，经常在上课的时候忽然暴躁，大喊大叫。而且不是一次两次，基本上你一个学期能遇到7、8次。而且很多时候并不是说很大的触怒了他，而且他特别敏感，稍微哪个地方他主观觉得不满意就开始暴怒让你按照他的意愿行事。而且你也千万不能有一点点不同意他的想法。
  2. GPA不仅仅非常非常低，而且乱给，而且十分主观。高二上还好，因为都是考试，你考多少分就是多少分，也没办法改变。但是高二下没有几次考试了就导致非常的主观，基本上就是看心情给分。所以你就会在班里看到一堆人天天当舔狗然后不停的混脸熟然后，基本上就是你跟他越熟他给你分越高。
  3. GPA 非常非常的低，平均分低到比另外一个老师低10几分！！！而且A+的极其之少，简直无法理喻，无法评价。
  4. Advanced简直就是主观评分的高二下的加强版，更加主观评分，更加无语和**。

无建议笑死 及时整理知识点，梳理框架什么的，自己脑子里的知识点不清晰的话会觉得学起来很痛苦，会越学越乱的。如果是没有别的可选了，想凑门ap可以考虑，如果不缺ap且不是很感兴趣的个人认为宁愿不学哈哈哈（碰到Rush只能祝您好运了，准备自学/课外学吧）

社会科学双经economics 两门ap考试的课
整体难度适中 社科方向基本上必选课程 但是课程进度有点紧（我们是他今年第一次教 可能时间规划没搞好但明年应该会好点
ED Eduardo教 人挺好而且很负责 不过课教的个人感觉没那么好但也可以
整体作业量不大 考试也不是特别多而且考的比较基础 考试分数占比大但是没有curve 刚接触经济方向的可能需要投入较多时间理解和练习来拿到高分
ap考试难度还可以 满分率挺高的 努努力拿两个满分问题不大
