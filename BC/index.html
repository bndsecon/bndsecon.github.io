Course Description

教授AP微积分BC的知识。
这门课主要取决于你是哪个老师，三个老师各有特色，在这里不适合进一步讨论。但Mr. Saju相对来说不错。高老师教授的BC内容比较难。

如果是Meth Devendra教的话，微积分BC 额 做好自学准备就行，自己得刷题练习。

Recommendation

如果你高一没学没考BC肯定得上，因为这门AP很重要，可以换很多学分，剩下好几万美刀，而且毕竟微积分理科基础嘛。如果高一考完了，建议不要选，因为GPA并不会因为这个更简单，而且可以花时间去学一些更有用的东西。毕竟做人嘛，稍微功力点挺好的，但是太。。也不行。（仅供参考）
