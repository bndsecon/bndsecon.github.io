CSA

Course Description

作为AP两门计算机课程中供给理科生的一门课。授课语言为Java。考试内容为选择题和手写编程题。其课程内容等同于AP CSA的考试大纲。

难度上来讲，如果你是一个偏文的人，而且此前从来没有接触过计算机，可能稍微有一些难度。但是由于CSA的考核更偏向于选择题（类似于NOIP初赛），所以只要背知识点然后刷题的话就可以取得接近满分。
