CSA

Course Description

作为AP两门计算机课程中供给理科生的一门课。授课语言为Java。考试内容为选择题和手写编程题。其课程内容等同于AP CSA的考试大纲。

难度上来讲，如果你不是一个理科生，而且此前从来没有接触过编程，可能稍微有一些难度。但是由于CSA的考核更偏向于选择题（类似于NOIP初赛），所以即使你编程能力差一点但是好好学习的话是可以学下来的。而且老师比较仁慈，不会像某些课一样压分或者留很多写半天的作业。

Comments

具体来讲就是一门正常的计算机科学课，从0起学，学习基本的Java基本语法到if else语句，for，while循环。数据类型，面向对象编程基础，数组，二维数组，动态数组，递归，继承，多态，插入排序选择排序，二分查找，等等。其对于面向对象：虽然java没有指针，但是其对于reference和object相关的考点很多。

从上课的角度来讲，CSA没有课后作业，所有的练习都在课上完成。下课之后只需要自行阅读课本即可，任务量相对较少。考试内容以每个月一次quiz，几次Lab（既当堂编写程序），和期末考试（有curve）为组成部分。

老师的话，Mr. Puneet是一个挺好的老师，十分的有意思，而且其实很和善的。期末的时候如果你GPA差一些的话他会根据你的学习情况给你一些鼓励。

csa的学习非常系统性，有成就感。学期初只能在老师的带领下单纯的复制语句，渐渐的理解更加深入，可以完全明白一个语句里每一个词的意义。最后普尼，无论大家怎么评价吧，我愿称之为我来十一五年最喜欢的老师之一。

Recommendation

如果你想学理工科的话就选CSA吧。
