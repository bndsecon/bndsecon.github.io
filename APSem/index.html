APSem

Course Description

是一门由Mr. Poth 教授的，以撰写论文为核心的课程。但是这个论文并不是专业学术论文，是更偏向于非专业性的学术论文。其上课内容主要为自习（存在争议），学习方法为自学（存在争议），作业量较多。

Comments

怎么说，我不太推荐这个课，因为老师不教东西，纯自学，作业量也比较多，每学期两个essay，4000字左右，其中要查巨多文献并总结。虽然gpa比较好，但相对的花费时间也更多，而且5分率非常低

还好吧，两个学期要写总共四篇文章（1.2k+2k+1.2k+2k）每篇要写7篇左右的资料阅读感受，各200左右.每篇文章要写一个outline（300-700），一个rough draft和一个final draft.要交的两个学期总共就这么多.分散到每天就一小会，文章堆到最后一天写的话大概是从放学写到晚12点的样子.给分挺松的，基本上按时交，交的是大概是对的内容就基本上不会被扣下a+.然后有4个presentation要做（两个10min的小组，两个8min的个人两个学期总共就这点事.一半上学期，一半下学期，下学期的记ap分

一个学期两篇paper（分别1200字和2000字）和两次演讲 上半学期的算在GPA里 下半学期的要算在AP成绩里. 如果只是想找个轻松的课上一上，不建议来，因为任务量时不时地会很大。如果真的是想提前学学写大学里经常会写的那种paper 可以考虑来 Poth就是按照大学paper标准讲的
