CSP

Course Description

是一门相对来说并不是很硬核的计算机课，基本以背概念为主。也会学习一点编程，但是学到基本的加减乘除，输入输出就不学了。
如果你不想成为硬核理工人的话也可以学。

AP的话，在学期中的时候会以小组为单位制作一个小程序，然后撰写报告。AP成绩最后会由这个报告和考试的综合为评判标准，5分难度相对较高。

很多人是因为看了学校官方的选课讲座底下的课程推荐，未尝不可，但是其实并不能带给你很大的帮助。

老师的话，Mr. Puneet是一个挺好的老师，十分的有意思，而且其实很和善的。期末的时候如果你GPA差一些的话他会根据你的学习情况给你一些鼓励。

但是老师并不是很推荐CSP，因为CSP和computer science的关系太少了，或者说，你很难说CSP是一门计算机课。。。
