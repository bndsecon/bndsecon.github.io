CSP

Course Description

是一门相对来说并不是很硬核的计算机课，以了解为主。也会学习一些编程，但是学的比CSA要浅一些。
如果你不想成为硬核理工人的话也可以学。

AP的话，在学期中的时候会以小组为单位制作一个小程序，然后撰写报告。AP成绩最后会由这个报告和考试的综合为评判标准，5分难度相对高一点。

很多想要学人文学科的同学是因为看了学校官方的选课讲座底下的课程推荐才选的CSP。虽然说有帮助，但是对人文学科的学习帮助并不是很大。

老师的话，Mr. Puneet是一个挺好的老师，比较和善的。
