Cal3

Course Description

作为十一学校提供的最高级的数学分析课，其难度和很大的作用使其成为了高二硬核理工科学生的不二之选。任课老师高老师作为正宗美国数学博士，教的很好，而且也很清晰，很多看起来很难的内容实际学起来并不是很难。如果你想学习计算机或者数学，那么Cal3是你高二的不二之选。如果你想学习物理，那么你必须要选择Cal3，要不然就等同于加减乘除还不会算就开始学习奥数。

Cal3最主要的作用是为理工类课程的完整世界提供一个相对基础的数学基础，比如如果你连偏导数都不学你根本没有学习机器学习、物理、统计的基础。此外，作为AP物理C力学和电磁学的数学基础，如果你并没有学过Cal3，直接选择物理C基本上什么都听不懂。物理中物理某物体沿路径在向量场中做功，流体的散度，电磁学中一点的通量，都是在Calculus3曲线曲面积分的基础上学习的。

课程内容可以参考任何Calculus 3的Syllabus，教材：CALCULUS EARLY TRANSCENDENTALS。 Calculus3是很多美高都会开设的多元微积分课程，其课程内容如下：

  • Vector and the Geometry of Space
  • Vector Functions
  • Partial Derivatives
  • Multiple Integrals
  • Vector Calculus

从三维空间的解析几何开始，学习向量，叉乘点乘的深入学习，基础的线性代数和空间几何体的解析算法。
在学习完解析几何后，开始学习曲线论。在学习完曲线论，开始学习多元函数和多元函数的偏导数，方向导数和偏导数，最后学习梯度和拉格朗日乘子法。
在学习完多元函数的微分之后，就进入多元微积分的重头戏：多元函数的积分。从简单的矩形二重积分开始，学习General形式的二重三重积分。在学习完普通坐标系的二重三重积分过后，开始学习柱坐标、球坐标下的三重积分。在学习完三重积分过后，开始学习二重三重积分在物理学中的应用，最后，学习以线性代数结合的空间坐标系变换下的多重积分。
在学习完多重积分后，开始学习空间下的多元曲线和曲面积分，曲线积分的基本定理，第一型第二型曲线积分，第一型第二型曲面积分。然后学习散度和旋度，最后以斯托克斯公式、散度定理为结尾。

Recommendation

这门课程虽然内容很难，但是其成绩大部分都是靠高老师仁慈的Curve上来的。课程会将平均分Curve到92，期末考试的Curve会在20分左右。作业的话是平均两周一次作业，内容为实际练习题，基本为学习的定理的应用。
