Course Description

作为两门数学AP中的一门，其应用型吸引了大量想学习泛STEM专业和非STEM专业的同学的青睐。其课程内容大体为统计图的识别，最小二乘线性回归，实验与抽样，随机分析，概率，概率密度分布，抽样分布，假设检验，线性回归模型。
其应用范围很广，而且课程难度适中，不要求硬核的数学知识背景。值得注意的是，AP 统计学要求一定的英语能力，因为其阅读提取信息和大段文字的书写不同与其他理科，其对于语言能力的要求更高。
GPA规则：平时没有作业，每个单元有一次考试，大概一个月考一次。每次考试70分以上记满分，然后依次递减。没有考满分的同学可以选择在出分之后提交作业来获得一定程度的加分。最后期末考试不Curve。整个年级大约有数十个A+。
