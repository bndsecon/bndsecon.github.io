Course Description

讲授政治科学和法律的一门课程，授课老师专业知识很丰富很强，而且为人友善，亲和力非常强。如果你想学习社会科学或者法律的话非常推荐，授课老师北大毕业而且在美国读的法学研究生，还有纽约州律师执照。也可以咨询她关于相关的事宜。
