APBio

Course Description

AP理科当中最难的一门AP课。由于其特殊性和大家对“医学”，“生物学”等专业的向往，一代又一代的学生们都走向了学习AP Biology的道路。

其任课老师Mr. Philip Coverson是AP项目甚至整个十一国际部最好的理科老师，为人和善热情，极受欢迎。

AP bio
前置要求：G10 biology（先修班，需要考试入班）或biology honors（向G11，G12全体同学开放）

课程“含金量”：对想要major biology的同学来讲，APbio的五分还是很重要的，因此极力推荐G10的时候上bio，这样G11就能学AP，相对来讲五分压力会小很多。当然自学五分也是一种选择，身边很多朋友也这么整了，但翻车风险也存在。（某位很nb的学长就在这跌了一跤）

老师：
四节课是Philip Coverson上，他是AP项目理科最好老师没有之一。会合理延ddl并且在申请季给学生放假，会有一些神奇的加分项（比如看球赛打卡）。
一节课丁冠乔老师上，不过出于种种原因他的课会被拿来做一下“相对不重要”的事情（譬如讲do now，做retest甚至直接占掉做核酸）

作业：
每周一到两个worksheet，其实就是带着你总结一遍本周学习的知识点。worksheet应该都是网上找的，题不难但部分情况下体量也不小。每个worksheet需要一到三小时完成。
特别提醒：一些极其麻烦的表格总结/知识点梳理题会和G10的project重合，理论上可以直接套！课上记录电子版笔记对做作业有特别大的帮助！

另一种作业形式是discussion，选择一个给出的问题，写出自己的思考，150+词。同时需要给别人写两条至少75字的回复。收到5条回复会获得额外加分。
一般来讲1-1.5个月一次，但实际上本学年目前就来了两次，后面coverson就没提这茬了。

考试：
原则上每周有2-3个do now，但实际上没学新课就不会有所以频率也不会高到离谱。形式为5道AP选择题，计时完成，时间会从十分钟逐渐缩短到八分钟。每组（2-3个do now为一组）去除一个最低分然后登成绩。由于AP大部分是情景题因此不会像honors那么要求死记硬背，但心理压力还是很大的。

test一个学期三四次，每次45min，20道AP选择题+1道简答题。
curve以retest的形式进行，retest一般为10道AP选择题+1道简答题，也是一节课完成。根据百分比得分可以补回最多50%-30%（随课程进行减少）的考试扣分（由于聪明的大家考得都很好所以retest的curve颇有打发狗的意味）

期末考试看情况吧，就考过一次不好说，有curve但方式挺阴间的。

GPA：
认真听课不可能下A，G11同学普遍比G12同学高5分左右（懂得都懂）但A+确实需要冲一冲。APbio不是死记硬背的课！拿分要理解然后推理！
